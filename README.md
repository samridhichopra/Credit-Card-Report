# 💳 Credit Card Weekly Analysis Dashboard

## 📊 Project Overview

This project presents an **interactive Power BI dashboard** that analyzes **weekly credit card transactions and customer insights**.
The goal of this report is to monitor credit card performance, understand customer behavior, and identify trends that impact revenue generation.

The dashboard provides a **data-driven view of financial performance and customer segmentation**, helping businesses make better strategic decisions.

---

# 🚀 Dashboard Sections

The Power BI report consists of **two main dashboards**:

## 1️⃣ Credit Card Transaction Report

This dashboard focuses on **transaction-level analysis** and financial performance of credit card usage.

### 🔑 Key KPIs

* 💰 **Total Revenue**
* 💳 **Total Transaction Amount**
* 🔢 **Total Transaction Count**
* 📈 **Total Interest Earned**

### 📌 Insights Covered

* Weekly revenue trends
* Revenue distribution across transaction categories
* Spending patterns across different dimensions
* Overall financial performance of credit card transactions

---

## 2️⃣ Credit Card Customer Report

This dashboard focuses on **customer-level insights** to understand how different customer segments contribute to revenue.

### 🔑 Key KPIs

* 💰 **Revenue Generated**
* 👤 **Customer Income**
* ⭐ **Customer Satisfaction Score**
* 📈 **Interest Earned**

### 📌 Insights Covered

* Weekly revenue contribution by customers
* Revenue distribution across **age groups**
* Customer segmentation by **education level**
* Demographic insights influencing credit card usage

---

# 🗂 Dataset Information

The project uses datasets containing **credit card transaction details and customer information**.

These datasets include attributes such as:

* Transaction amount
* Revenue generated
* Customer demographics
* Interest earned
* Customer income and satisfaction score

The data is used to generate meaningful **KPIs, trends, and segmentation insights** in Power BI.
